# السبورة — دليل معلم الرياضيات الرقمي

موقع داخلي للمعلمين: يختار المعلم المادة → الصف → المسار (إن وجد) → الدرس،
فيصل لصفحة تحضير إلكتروني كاملة مع روابط أدوات تفاعلية (Desmos, GeoGebra, PhET...).

لا توجد قاعدة بيانات: كل المحتوى عبارة عن ملف بيانات واحد يمكن تعديله مباشرة.

## التشغيل محليًا

```bash
npm install
npm run dev
```

يفتح المشروع تلقائيًا على `http://localhost:5173`.

للبناء للنشر:

```bash
npm run build
```

الناتج يكون في مجلد `dist/` — يمكن رفعه على أي استضافة ثابتة (Vercel, Netlify, GitHub Pages...).

## إضافة درس جديد

كل المحتوى في ملف واحد: `src/data/curriculum.ts`.

1. افتح الملف وابحث عن الصف/المسار المناسب.
2. أضف عنصرًا جديدًا داخل مصفوفة `lessons` بنفس شكل الدروس الموجودة:

```ts
{
  slug: "unique-lesson-slug",       // يُستخدم في رابط الصفحة
  title: "عنوان الدرس",
  unit: "اسم الوحدة",
  duration: "حصة واحدة (45 دقيقة)",
  objectives: ["هدف 1", "هدف 2"],
  prerequisites: ["متطلب سابق"],
  sections: [
    { title: "تمهيد", body: ["فقرة أولى", "فقرة ثانية"] },
    { title: "العرض", body: ["..."] },
  ],
  tools: [
    {
      name: "اسم الأداة",
      url: "https://...",
      icon: "graph", // أو: simulation, geometry, video, worksheet, board, link
      note: "شرح مختصر لكيفية استخدام الأداة في هذا الدرس",
    },
  ],
  homework: ["بند 1"],
}
```

الصفحة تُبنى تلقائيًا من هذه البيانات — لا حاجة لإنشاء أي ملف React جديد لكل درس.

## إضافة مادة دراسية جديدة

أضف عنصرًا جديدًا في نفس الملف داخل مصفوفة `curriculum` (على مستوى المادة)، واجعل
`available: true`، ثم أضف له أيقونة في `src/components/icons.tsx` داخل `subjectIconMap`.

## البنية التقنية

- **React + TypeScript + Vite**
- **React Router v7** (وضع البيانات — `createBrowserRouter`)
- **Tailwind CSS v4** للتنسيق
- **lucide-react** للأيقونات (بدون أي إيموجي)
- بدون أي قاعدة بيانات أو باك إند — كل المحتوى بيانات ثابتة في `src/data/curriculum.ts`

## الأيقونات المتاحة للأدوات (`icon` في كل أداة)

| القيمة | تُستخدم لـ |
|---|---|
| `graph` | آلات بيانية (Desmos وغيرها) |
| `simulation` | محاكيات (PhET وغيرها) |
| `geometry` | أدوات هندسية (GeoGebra وغيرها) |
| `video` | فيديوهات شرح |
| `worksheet` | أوراق عمل قابلة للطباعة |
| `board` | سبورة تفاعلية |
| `link` | أي رابط آخر عام |
